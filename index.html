Connect & Support

Grease-Pencil Style Drawing for 3ds Max

GracePencil is a native C++ plugin for Autodesk 3ds Max that brings a complete Blender-Grease-Pencil–inspired 2D/3D drawing system directly into the Max viewport. It lives as a standard scene object — draw on any plane in 3D space, build frame-by-frame animation, apply per-layer visual effects, and manage deep multi-layer compositions, all without leaving Max.

Multi-Layer System

Unlimited layers with blend modes, opacity, clipping masks, group folders and per-layer effects.

Frame-Based Animation

Keyframes, onion skinning, auto-key, layer interpolation/tween, and animated layer transforms.

Pen Tablet & Touch

Full Wacom / Windows Ink pressure sensitivity. Pressure drives stroke width and opacity in real time. Touch gestures supported.

Photoshop Brush Import

Import any .abr brush set. PNG stamps cached persistently in %TEMP%\MaxPencil — no re-import on next session.

10 Brush Types

Solid Stroke, Solid Fill, Square, Dots, Grunge, Grain, Marker, Watercolor, Map and Custom Stamp. Each with full width taper control.

Visual Effects

Per-layer Drop Shadow, Outline (inner/outer), Self-illumination Glow, Override Color and Wiggle hand-drawn animation.

3D Drawing Planes

Draw on View, Front, Top, Left, Custom or Pinned (face-view) planes. Full position, yaw, pitch spinners with live grid display.

Max Native Integration

Standard scene object — fully compatible with Max materials, renderer, cameras, and the modifier stack. Saves inside .max files.

Use Cases & Creative Applications

GracePencil is not a single-purpose tool — it fills a wide range of roles depending on your pipeline and artistic goals.

Pre-Production & Scene Planning

Place a GracePencil object in your scene and sketch directly on top of your 3D environment to plan compositions, block out camera cuts and mark areas of interest — without touching any geometry.

  • Guide lines & annotations: Draw proportion guides, center lines, golden-ratio grids or written notes that float in 3D space as visual reminders while modelling or lighting.
  • Layout blocking: Rough-sketch character silhouettes or set-piece positions on a View plane before committing geometry to the scene.
  • Camera path sketching: Draw the intended camera trajectory directly in the viewport to share with a director or client before animation begins.

2D Animation Living in 3D Space

GracePencil strokes exist as real 3D geometry, so they respond to 3D cameras, lighting and depth of field. This enables a class of effects popularised by studios like Arkane Studios (Dishonored, Deathloop) — hand-drawn 2D elements that are anchored in 3D world space:

  • Impact & hit effects: Frame-by-frame drawn flashes, speed lines and energy bursts that follow 3D object positions via Object Tracker.
  • Magic / VFX overlays: Looping hand-drawn rune animations, summoning circles or stylised fire/smoke drawn on a billboard plane that faces the camera at all times (View plane mode).
  • Dialogue & UI elements: In-world sketched UI panels, damage indicators, or comic-book speech bubbles placed in scene space and animated with keyframes.
  • Retro / lo-fi motion: Deliberately choppy hand-drawn motion using the Wiggle system to simulate cel-animation jitter on top of 3D renders.

Stylized 2D + 3D Hybrid Rendering

Combine GracePencil stroke layers with standard 3ds Max geometry to build a mixed-media stylized aesthetic — the same visual language used in games like Borderlands, Hi-Fi Rush, or Guilty Gear Strive:

  • Ink outlines on 3D models: Draw precise ink-style outlines around 3D objects on a surface-projected layer. The Object Tracker keeps them attached as the model animates.
  • Crosshatching & shading: Hand-draw tonal shading passes on top of 3D renders to create a graphic-novel illustration effect.
  • Flat colour fills: Use Solid Fill layers with Blend Mode set to Multiply or Overlay to add cel-shaded colour passes over existing materials.
  • Stylised FX: Outline-style explosions, dust clouds or water splashes drawn in 2D but tracked to 3D particle emitter positions.

Concept Art & Storyboarding Inside Max

Artists can stay inside 3ds Max for the full ideation-to-production pipeline without switching to an external paint application:

  • Rough sketching directly over geometry: Enable Draw On Surface and sketch anatomy, costume or architectural detail directly on your 3D mesh — the strokes conform to the surface.
  • Storyboard panels: Use multiple keyframes on separate layers to build a rough animatic storyboard complete with camera moves, entirely inside a single .max file.
  • Texture & UV annotations: Mark UV seams, texture tile boundaries or important surface details as guide strokes that render into reference sheets.

Pen Tablet & Touch Workflow

Pressure Sensitivity

Works with any Wacom, Huion, XP-Pen or Windows Ink compatible device. Pen pressure simultaneously controls stroke width (via the Width envelope) and opacity (via Min Opacity). Light touch = thin, transparent strokes; full pressure = bold, opaque marks.

Touch Input

Windows touch events are supported for tablets and touch-screen monitors. Draw with your finger or stylus in the viewport. The Smooth parameter cleans up the natural wobble of freehand touch input into clean curves automatically after each stroke.

Pen Tilt & Speed

Some brush types (Watercolor, Grunge, Grain) respond to the natural variation in drawing speed, giving slower strokes more buildup and faster strokes a lighter feel — mimicking real traditional media behaviour without requiring a tilt-capable device.

Sub-Object Level

GracePencil exposes a sub-object level in the 3ds Max modifier stack, giving you access to the underlying stroke geometry for selection and transformation — exactly like working with an Editable Spline or Poly.

How to access: Select the GracePencil object → go to the Modify panel → click the sub-object level icon in the modifier stack, or activate the Select tool in the Grace Tools rollup. The toolbar also exposes quick select access.

S

Stroke Selection

At sub-object level you can box-select one or more complete strokes in the viewport. Selected strokes highlight in a distinct colour to confirm the selection.

• Click a stroke to select it (deselects others).

• Ctrl+Click to add or remove strokes from the selection.

• Drag a region box to select all strokes inside it.

• Delete key removes selected strokes from the current frame.

T

Transform Selected Strokes

With strokes selected you can use the standard Max transform gizmos:

• Move (W) — translate strokes freely in 3D space or snap to the drawing plane.

• Rotate (E) — rotate strokes around the selection centroid.

• Scale (R) — uniformly or non-uniformly scale strokes.

Transformations are non-destructive: the stroke point positions update live in the viewport, and the change is recorded as an undo step.

P

Per-Stroke Properties

Once strokes are selected, the Brush & Colors rollup updates to show the properties of the selected strokes:

• Apply Settings to Selected — stamps the current brush settings (type, width, color, opacity, stamp params) onto every selected stroke at once.

• Color swatch clicks in the rollup propagate to selected strokes in real time.

• Width Start/Mid/End changes reshape selected strokes without redrawing.

Erase at Sub-Object Level

The Erase tool operates at sub-object level when active:

• Stroke mode — click or drag the eraser brush to delete entire strokes that fall within the brush radius.

• Point mode — surgically cuts individual points from strokes, splitting them into multiple shorter strokes.

• Region modes — Rectangle, Ellipse, Lasso or Polygon region constraints define the exact area of erasure, giving precise control over which parts of a drawing to remove.

Smudge & Relax at Sub-Object Level

Smudge and Relax also work as sub-object-level sculpting operations on stroke points:

• Smudge pushes stroke points in the direction of your drag, bending and distorting existing strokes without redrawing them — like smearing ink while still wet.

• Relax iteratively averages the position of points within the brush radius, removing high-frequency noise and smoothing jagged freehand strokes into flowing curves.

• Both tools respect the Tool Scope setting — restrict them to the active layer or let them operate across all visible unlocked layers simultaneously.

1. Grace Tools Rollup

The main drawing tools rollup. All primary interaction modes live here.

Drawing Tools

Draw

Left-drag to paint strokes on the active layer using the current brush settings. Pressure-sensitive when a pen tablet is connected.

Erase

Removes stroke geometry. Works in Point or Stroke mode, with optional region selection (Rectangle / Ellipse / Lasso / Polygon).

Select

Box-select individual strokes to move, scale or delete them. Supports multi-selection.

Smudge

Drag stroke points in the direction of brush motion — like smearing wet ink. Operates on all layers or the selected layer only (Tool Scope).

Relax

Smooths and relaxes stroke curvature under the brush radius without destroying the overall shape.

Gradient

Drag to place a gradient fill layer (Linear or Radial). Shift+drag edits an existing gradient stroke.

Bucket

Flood-fills the nearest closed outline region. Adjust Gap tolerance to close small openings.

Shape

Draws parametric shapes: Ellipse, Rectangle, Hexagon, Star, Egg, Arrow, Gear, Parallelogram. Can target a selected shape layer or always create new.

Bucket Fill

Gap — Tolerance (in scene units) for closing small gaps in outlines before flood-filling. Increase if the fill leaks.

Erase Options

Mode Stroke — deletes the entire stroke when any of its points enter the brush radius.
Point — surgically removes only the individual points under the brush and splits surviving runs into separate strokes.

Region — Constrains the erase area shape: Rectangle, Ellipse, Lasso (free-hand), Polygon (sharp corners).

Shape Drawer

Type — Shape template: Ellipse, Rectangle, Hexagon, Star, Egg, Arrow, Gear, Parallelogram.

Closed Shape — Forces the generated stroke to be a closed loop.

Shape Layer — When checked, dedicated shape layers are used for better organisation.

Draw Mode — Selected Shape Layer: draws into the current layer. Always Make New: creates a fresh layer for each shape.

Tool Scope

All Layers — Smudge, Relax, and Erase affect every visible, unlocked layer simultaneously.
Selected Layer Only — Restricts those tools to operate exclusively on the currently active layer.

2. Draw Options Rollup

Controls the 3D plane on which strokes are painted. Every stroke is recorded in world-space on a plane you define here.

Drawing Plane

Plane — Selects the plane preset:
View — always faces the active camera/viewport.
Front / Top / Left — world-aligned axis planes.
Custom — manually positioned via X, Y, Z and Yaw/Pitch spinners.
Pinned (face view) — locks to the current camera angle at the moment of activation.

X / Y / Z — World-space position of the custom drawing plane origin.

Yaw — Horizontal rotation of the custom plane (degrees).

Pitch — Vertical tilt of the custom plane (degrees).

Center to View — Teleports the plane center to the current viewport camera target.

Pick Node — Aligns the drawing plane to the pivot and orientation of a picked scene object.

Show Grid — Draws a visual reference grid on the active drawing plane in the viewport.

Surface & Symmetry

Draw On Surface — Projects every stroke point onto the nearest surface of scene geometry as you draw (surface snapping).

Offset — Lifts projected points off the surface by this distance in scene units (prevents Z-fighting).

Depth — Limits surface projection to geometry within this depth from the drawing plane.

Symmetry

X / Y / Z — Enables live mirror symmetry along the chosen world axis. Each stroke is mirrored in real time.

Pick — Pick a node whose pivot defines the symmetry origin and orientation.

Origin label — Displays the name of the currently picked symmetry node ("Origin" if none picked).

3. Brush & Colors Rollup

Every aspect of how strokes look and feel — from pixel width to custom Photoshop stamp brushes.

Width Profile

Controls the stroke width envelope — how thick the stroke is at its start, middle and end.

Start — Width multiplier at the very first point of the stroke.

Mid — Width multiplier at the midpoint of the stroke.

End — Width multiplier at the last point of the stroke.

Smooth — Number of smoothing passes applied to the stroke path after drawing. Higher values = cleaner curves.

Fill Closed — Automatically fills the interior of any closed stroke with the Fill color.

Paint Color

Stroke color — The primary outline/edge color of the stroke.

Fill color — The interior fill color used when Fill Closed is on or brush type is Solid Fill.

Opacity — Global opacity of new strokes (0–100%). Blends with layer opacity.

Pressure (Pen / Touch)

Pressure Sensitivity — Enables pen-tablet or touch pressure to modulate stroke width dynamically while drawing.

Min Opacity — The opacity floor at the lightest pen pressure (0–1). Prevents fully transparent strokes.

Map Brush & .abr Import

Assign any bitmap or Photoshop brush set as the stamp texture for Custom / Stamp mode.

Pick… — Opens the 3ds Max material browser to select any BitmapTex as the stamp image.

X (Clear) — Removes the current stamp map and resets to solid-stroke rendering.

Import .abr — Opens the Photoshop brush importer. Parses the .abr file, shows a progress bar with a Cancel button, and saves all extracted PNG stamps to %TEMP%\MaxPencil\<name>_brushes\ so they persist between sessions.

Map name label — Shows the filename of the active stamp map.

Tile U / V — Tiling multipliers for the stamp texture coordinates.

Apply Settings to Selected — Copies all current brush settings (type, width, color, stamp params) to every selected stroke in the viewport.

Custom Stamp Options

Fine-grained control over how the stamp image is distributed along the stroke path. Visible only when Brush Type is Custom / Stamp or Map.

Spacing

Distance between successive stamp placements along the stroke (scene units).

Angle

Fixed rotation of each stamp instance in degrees.

Follow Stroke Direction

Rotates each stamp to align with the local tangent direction of the stroke.

Falloff

Alpha falloff from the center of each stamp to its edges (0 = hard, 1 = fully soft).

Scatter

Random positional offset applied perpendicular to the stroke path.

Size Jitter

Random variation in the scale of each stamp instance.

Angle Jitter

Random rotation variation applied on top of the base Angle setting.

Colorize

Tints the stamp grayscale pixels with the active stroke color.

A live preview of the stamp bitmap is displayed below the options panel.

4. Layers Rollup

Non-destructive, stackable layers — the backbone of the GracePencil workflow.

Layer List & Management

The list shows all layers and group folders in render order (top = drawn last = on top).

Add — Creates a new empty drawing layer above the current selection.

Del — Permanently deletes the selected layer(s) and all their strokes.

Up / Dn — Moves the selected layer one position up or down in the stack.

+Grp — Creates a new empty group folder at the top of the stack.

Grp — Moves selected layers into a new group folder.

Ungr — Extracts selected layers out of their group.

Fold — Collapses or expands the selected group folder in the list.

◳ (Float) — Detaches the Layers rollup into a floating window.

Layer Options

Visible — Toggles the layer's visibility in the viewport and at render time.

Lock — Prevents any edits to the layer's strokes while keeping it visible.

Onion — Includes this layer in the onion-skin ghost display.

Interpolate — Enables smooth continuous tween morphing between this layer's keyframes.

Blend Mode & Opacity

Blend — Compositing mode for this layer: Regular, Multiply, Screen, Overlay, Add, Subtract, and more.

Opacity — Overall layer opacity (0–1). Multiplies with individual stroke opacity.

Clipping Mask

Clip to — Selects a layer whose strokes act as the alpha mask for this layer. Only pixels inside the base layer's painted area will be visible.

Inv — Inverts the clipping mask (shows outside the base layer instead).

5. Animation & Keys Rollup

Build frame-by-frame animation and control onion skinning directly from this rollup.

Keyframe Controls

+Key — Inserts a new blank keyframe on the active layer at the current timeline position.

Dup — Duplicates the current keyframe's stroke data to a new key — useful for tweens or iterating on a drawing.

-Key — Deletes the keyframe at the current timeline position on the active layer.

<< Prev / Next >> — Jumps the timeline scrubber to the previous or next keyframe of the active layer.

Auto Key — When enabled, any stroke drawn automatically creates or overwrites a keyframe at the current frame. Disable to draw on a single persistent frame.

Clear Frame — Removes all strokes from the current frame of the active layer without deleting the key.

Onion Skinning

Displays ghosted previews of neighbouring frames as a drawing reference — essential for smooth animation.

Onion Skinning — Master toggle. Enables the ghost overlay in the viewport.

Before — Number of previous frames to show as ghosts (shown in a cooler colour).

After — Number of upcoming frames to show as ghosts (shown in a warmer colour).

Materials

Apply Materials — Regenerates and assigns 3ds Max materials to all stroke geometry in the scene based on their color, opacity and effect settings. Run this after changing colors or effects to see the result in renders.

6. Effects Rollup

Per-layer visual effects, color overrides, and the Wiggle hand-drawn animation system.

Drop Shadow

Enable — Activates the drop shadow effect for all strokes on this layer.

Offset X / Y — Horizontal and vertical displacement of the shadow from the original stroke (scene units).

Opacity — Transparency of the shadow (0–100%).

Color — Shadow tint color (default black).

Outline

Enable — Draws a coloured border around each stroke on this layer.

Width — Thickness of the outline in scene units.

Mode — Inner: border grows inward. Outer: border grows outward.

Color — Outline tint color.

Glow (Self-Illumination)

Enable — Assigns a self-illumination material component that makes strokes appear to emit light in renders.

Intensity — Strength of the glow emission (0–100). At 100 the stroke is fully self-lit regardless of scene lighting.

Override Color

Stroke — Forces all strokes on this layer to render with this color, regardless of their individual stroke color setting.

Fill — Forces the fill color of all strokes on this layer.

Override with Layer Color — Master switch. When enabled, the two swatches above replace per-stroke colors. When disabled, individual stroke colors are used.

Wiggle (Hand-Drawn Animation)

Adds a procedural, time-varying jitter to stroke point positions — mimicking the natural unsteadiness of hand-drawn frame-by-frame animation.

Wiggle (hand-drawn) — Enables the wiggle effect on this layer.

Str (Strength) — Maximum spatial displacement of each point in scene units. Higher = more chaotic movement.

Rate — Speed of the oscillation in cycles per second. Higher = faster wobble.

Object Tracker

Binds the layer's strokes to a scene object so they follow it through the animation without parenting the GracePencil object itself.

Track Object — Enables tracking. All strokes on this layer are transformed in world-space by the tracked node's transform delta.

Pick — Click then click a scene object to assign it as the tracked node.

Node label — Displays the name of the currently tracked node ("None" if no node is assigned).

Tip: All effects except Wiggle require you to press Apply Materials (in the Animation rollup) to propagate the changes to rendered materials. Wiggle and Object Tracker update in real time in the viewport.

7. Gradient Rollup

Creates and edits gradient fill strokes. Activate the Gradient tool then drag in the viewport to place a gradient layer. Shift+drag edits an existing gradient stroke.

Gradient Settings

Direction — Linear: straight colour ramp. Radial: circular burst from center to edge.

Width — Length/radius of the gradient band in scene units.

Color Stops

The gradient bar is an interactive control — click on it to select a color stop; drag stops to reposition them.

Add — Inserts a new color stop at the midpoint between two existing stops.

Del — Removes the currently selected color stop (minimum of 2 stops required).

Pos % — Precise position of the selected stop along the gradient (0 = start, 100 = end).

Color — Color of the selected stop. Click the swatch to open the Max color picker.

Plane Transform

Adjusts the 3D transform of the gradient plane after placement.

Offset X/Y/Z — Translates the gradient plane origin in world space.

Scale X/Y — Scales the gradient plane along its local U and V axes.

Rot X/Y/Z — Rotates the gradient plane around each world axis.

UV Transform

Independently adjusts the texture-space mapping of the gradient colour ramp.

Offset U/V — Shifts the colour ramp along U or V within the plane.

Scale U/V — Stretches or compresses the colour ramp.

Rotate — Rotates the colour ramp within the UV space of the plane.

8. Palette Rollup

A persistent colour swatch library that lets you save, organise and quickly apply colours during drawing sessions.

Saved Swatches

Apply to — Determines which color slot a clicked swatch fills: Stroke Color, Fill Color, or Both.

The Saved swatches grid shows all colours you have stored. Click any swatch to apply it to the target slot.

Add — Saves the current active Stroke color to the palette.

Clear — Removes all saved swatches after a confirmation prompt.

Recent Colors

The Recent strip automatically records the last colours used while drawing — no manual saving needed. Click any chip to reapply it.

Import / Export

Import — Loads a previously exported palette file (.gpl or GracePencil palette format) and merges it with the current palette.

Export — Saves the entire saved-swatches palette to a file for sharing or backup.

9. Qt Toolbar

A compact, floating Qt-based toolbar that gives instant access to the most-used tools without needing to scroll through rollups. Toggle it with Show Qt Toolbar in the Display rollup.

Tool Buttons

Quick-toggle buttons for Draw, Erase, Select, Smudge, Relax, Gradient, Bucket and Shape — mirrors the Grace Tools rollup. Clicking a button activates that tool immediately.

Brush Size

An inline spinner and drag-bar let you adjust the brush radius in pixels without opening the Brush & Colors rollup. Changes sync bidirectionally with the rollup.

Color Swatches

Inline Stroke and Fill color swatches. Click to open the standard Max color picker. Changes are reflected live in all rollups and new strokes.

Layer Controls

The toolbar also exposes the most critical layer operations so you can stay in the viewport:

Active Layer — A dropdown listing all layers. Select to switch the active drawing layer.

Add Layer — Creates a new layer above the current one.

Visible toggle — Eye icon per layer row; click to show/hide without opening the rollup.

+Key / -Key — Insert or delete a keyframe at the current frame, directly from the toolbar.

10. Display & Settings Rollup

Controls the viewport icon and the Qt toolbar visibility, plus global settings.

Icon & Label

Show Icon — Displays a small GracePencil icon in the viewport at the object's pivot. Useful for selecting the object in busy scenes.

Show Name — Renders the object name as a label next to the icon in the viewport.

Icon Size — Pixel size of the viewport icon (8 – 256 px).

Move Icon — Activates a drag handle that lets you reposition the icon in the viewport independently of the object pivot.

Move Strokes With Icon — When enabled, dragging the icon also offsets all strokes by the same delta — useful for repositioning the entire drawing.

Toolbar & Settings

Show Qt Toolbar — Toggles the floating Qt toolbar on or off. The toolbar position is saved per-session.

Settings… — Opens the GracePencil Settings dialog where you can configure:

  • Default brush parameters for new sessions
  • Viewport drawing quality
  • Undo stack depth
  • Material generation options
  • Keyboard shortcut bindings
